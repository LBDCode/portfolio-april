brewIt API |

brewIt is a public beer recipe API. Sign up for an API key, and search or browse a curated database of beer recipes for your next homebrewing effort.

Recipedia |

Recipedia is a recipe finder and meal planning application that allows you to search the Edamam API for new, interesting recipes and add them to your weekely menu. Once you've added recipes, you can build and text/email a shopping list of ingredients.

Schedulr |

Schedulr is an appointment scheduling and notification application. Targeted at small businesses, it allows clients to book, review, update, and cancel appointments. It also allows customers to review appointments.

MyLibrary |

Interface with the GoogleBooks API to search for new books. Build your own library of books you've read or want to read.

Simpson's Clicky Game |

How good is your memory? Click each player's card once (and only once!). Can you click them all without duplicating a selection?

r/LearnPython Scraper |

A two-player rock, paper, scissors game with chat functionality. RPS mutiplayer uses Google Firebase's real-time database to track score, user, and chat data.
